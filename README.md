# 🔬Хакатон "Инженерная лаборатория" - Команда Хаббл 2026

> **Цель:** Создать прототип биореактора для культивирования водорослей и генерации гидрогелиевых структур методом электрохимической фабрикации.

## 🧠 О проекте

Проект разработан командой **"Хаббл"** за 27 часов хакатона. Реализована прошивка (Arduino) и десктопный интерфейс (WinForms) для демонстрации концепции биоинженерного устройства.

**Ключевые особенности:**
- 📡 Сбор данных с датчиков
- 🖥️ Визуализация и управление через GUI на C# / WinForms

## 🧪 Результат

Проект остаётся в статусе полурабочего прототипа. Дальнейшая доработка не планируется, код предоставлен как есть для архива и портфолио.
